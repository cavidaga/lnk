2) What we collect and why

  • Technical data: Our hosting/CDN may temporarily process your IP address, user agent, timestamp, and requested URLs in server/security logs to operate and secure the site.
  • Analytics (cookieless): We use Vercel Web Analytics (no cookies, short-lived request-derived identifiers, anonymised). We may also use Vercel Speed Insights for Core Web Vitals; metrics aren't tied to an individual or IP.
  • Fonts: Google Fonts (fonts.googleapis.com/gstatic) delivers typography; Google may receive your IP and browser details; typically no cookies are set.
  • User registration: When creating a developer account for API use, we collect email address, password (encrypted), and company information. This data is necessary for account management and API key creation.
  • API usage: We track API requests made with API keys (IP address, request time, functions used). This data is necessary for rate limiting enforcement and usage statistics generation.
  • Contact channels: If you email or use a form, we process the data you provide to reply. (Currently: Formspree integration on the About and API documentation pages.)

3) Legal bases (GDPR Art. 6)

  • Legitimate interests (6(1)(f)): running, securing, and improving the site, monitoring API usage.
  • Contract (6(1)(b)): responding to your request, providing API services.
  • Consent (6(1)(a)): only if explicitly requested (not used for analytics now).

4) Cookies & similar technologies

We do not set non-essential cookies for visitors. Developer accounts and admin/logged-in areas may use strictly necessary session/auth cookies. Because analytics is cookieless, a prior-consent banner is not required.

Functional storage: For the feedback feature, analysis history, and developer panel, we use your browser's local storage (localStorage) functionality. This data is only stored on your device and does not collect personal information. If we later add tools that set non-essential cookies, we'll obtain consent first and update this notice.

5) Retention

  • Server/CDN logs: retained for 30 days, then deleted or anonymised.
  • Analytics: aggregate/anonymous with short-lived identifiers (~24h) by Vercel.
  • Developer accounts: Retained while account is active. Data is completely deleted 30 days after account deletion.
  • API usage data: Usage statistics and request logs are retained for 90 days, then anonymised.
  • Correspondence: retained as needed to address your request and for record-keeping.

6) Sharing & processors

Trusted providers (“processors”) operate this site on our instructions: Vercel (hosting, Web Analytics, Speed Insights), email provider, and (if used) a form service.
