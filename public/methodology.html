Ümumi baxış və terminlər

Sistem məqalə mətnini analiz edir, konteksti nəzərə alır və nəticəni iki əsas ox üzərində vizuallaşdırır: etibarlılıq (y oxu, 0–100) və siyasi hakimiyyət meyli (x oxu, −5.0…+5.0). Əlavə olaraq sosial-mədəni meyl ayrıca bal kimi təqdim edilir.

Etibarlılıq
Mətnin faktlara söykənmə dərəcəsi, mənbə istifadəsi, başlıq–mətn uyğunluğu və səhvlik ehtimalı.
Siyasi hakimiyyət meyli
Hakimiyyət institutlarına yaxınlıq/uzaqlıq siqnalları, çərçivələmə və söz seçimi.
Sosial-mədəni meyl
Dəyərlər, kimlik və mədəni mövzularda meyllilik göstəricisi.

Ballar və şkalalar

  • Etibarlılıq (0–100): 0–39 zəif, 40–59 orta, 60–79 güclü, 80–100 yüksək etibarlılıq.
  • Siyasi hakimiyyət meyli (−5.0…+5.0): −5 kəskin müxalif çərçivə, 0 neytral, +5 güclü hökumətyönümlü çərçivə.
  • Sosial-mədəni meyl (−5.0…+5.0): Mənfi/pozitiv dəyərlər çərçivəsi və vurğular.

Qeyd: Ballar məqalənin daxilindəki siqnalların yığcam təsviridir; bu, müəllifin “tam siyasi mövqeyi” deyil.

Diaqnostik göstəricilər

  • Dilin yüklülüyü (0–100): emosional sözlər, hiperbola, təhqiramiz dil siqnalları.
  • Mənbə şəffaflığı (0–100): istinadların sayı, tipi, linklənməsi və şəffaflığı.
  • Başlığın dəqiqliyi (0–100): başlıq-məzmun uyğunluğu, klikbeyt ehtimalı.
  • Dil siqnalları: language_flags — ilə işarələnir, aşkar edilmiş konkret terminlər və kateqoriyaları göstərir.

Məqalə növü (meta)

meta.article_type sahəsi məzmunu “xəbər”, “rəy”, “analitika”, “reportaj” və s. kimi təsnif edir. Bu, nəticələrin interpretasiyasında mühüm kontekstdir (məsələn, rəy yazılarında dil yüklülüyü təbii olaraq daha yüksək ola bilər).

JSON çıxarış sxemi

Niyə JSON?

JSON (JavaScript Object Notation) — verilənlərin strukturlaşdırılması üçün geniş istifadə olunan formatdır. Sadə və yüngül sintaksisə malikdir, həm insanlar, həm də kompüterlər tərəfindən rahat oxunur.

LNK.az nəticələri JSON formatında bu səbəblərdən təqdim edir:

  • Şəffaflıq: Bütün göstəricilər açıq və standartlaşdırılmış quruluşda göstərilir.
  • Avtomatlaşdırma: JSON digər sistemlər tərəfindən asanlıqla oxunur və emal olunur.
  • Uyğunluq: JSON formatı istənilən proqramlaşdırma dili və platforma ilə inteqrasiya oluna bilər.

Beləliklə, JSON çıxarışı yalnız vizual nəticəni göstərmir, həm də araşdırmaçılar, jurnalistlər və tədqiqatçılar üçün təhlil nəticələrini digər vasitələrə daşımağa imkan verir.

Nümunə sxem (Azərbaycanca mətn tələbi ilə):

{
  "meta": {
    "article_type": "xəbər | rəy | analitika | ... "
  },
  "scores": {
    "reliability": {
      "value": 0-100,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf. Vergüllərlə cümlələri ayırma."
    },
    "socio_cultural_bias": {
      "value": -5.0 ... 5.0,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf."
    },
    "political_establishment_bias": {
      "value": -5.0 ... 5.0,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf."
    }
  },
  "diagnostics": {
    "language_loadedness": 0-100,
    "sourcing_transparency": 0-100,
    "headline_accuracy": 0-100,
    "language_flags": [
      { "term": "…", "category": "…"}
    ]
  },
  "cited_sources": [
    { "name": "…", "role": "…", "stance": "…"}
  ],
  "human_summary": "Aydın, axıcı Azərbaycan dilində xülasə."
}

Analiz prosesi

  1. Məlumatın toplanması: İstifadəçi URL daxil edir; sistem məzmunu götürür və təmizləyir.
  2. Analiz: Model mətni qiymətləndirir, balları və əsaslandırmaları çıxarır.
  3. Strukturlaşdırma: Nəticələr JSON obyektində qaydaya salınır.
  4. Vizualizasiya: Nəticə xəritə (koordinat oxları) və diaqnostika kartları ilə təqdim edilir.
  5. Arxivləşdirmə: Ətrafınızla paylaşım üçün link yaradılır (əgər aktivdirsə).

Qrafiki necə şərh etməli

LNK.az-da istifadə olunan koordinat oxları sistemi Ad Fontes Media Bias Chart modelindən ilhamlanmışdır. ABŞ-da yaradılan bu model media mənbələrinin etibarlılıq və ideoloji meyl oxları üzərində yerləşdirilməsi ilə tanınır.

Biz bu yanaşmanı Azərbaycan mediasının reallıqlarına uyğunlaşdıraraq, xüsusi olaraq siyasi hakimiyyət meyli və sosial-mədəni meyl kimi parametrlərlə genişləndirdik. Məqsədimiz beynəlxalq praktikadan faydalanaraq, yerli kontekstdə daha dəqiq və faydalı qiymətləndirmə təqdim etməkdir.

  • Yuxarı daha yüksək etibarlılıq, aşağı daha zəif etibarlılıq.
  • Sol müxalif çərçivə siqnalları, sağ hökumətyönümlü siqnallar.
  • Tək bir nöqtə bütün mətni ümumiləşdirən siqnalın ortalamasıdır; içindəki müxtəlif səslər “dil siqnalları” bölməsində görünə bilər.

Məhdudiyyətlər və məsuliyyət qeydi

  • Nəticələr proqnoz xarakterlidir və məqalənin versiyasından asılıdır.
  • Parça-parça sitatlarla zəngin, ironiyalı və ya kontekstdən asılı mətnlərdə səhv dərəcəsi arta bilər.
  • Məqalələri oxuduqdan sonra bu alətlə kifayətlənməmək və bir neçə mənbə ilə yoxlamaq tövsiyə olunur.

Etika və ədalət prinsipləri

  • Şəffaflıq: sxem, meyarlar və əsaslandırmalar göstərilir.
  • Müstəqillik: siyasi və ya maliyyə asılılığımız yoxdur. Lakin layihə ianələr üçün açıqdır.
  • Məlumat məxfiliyi: toplanmış məlumat yalnız təhlil və təkmilləşdirmə üçündür.
