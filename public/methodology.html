Ümumi baxış və terminlər

Sistem məqalə mətnini analiz edir, konteksti nəzərə alır və nəticəni iki əsas ox üzərində vizuallaşdırır: Etibarlılıq (y oxu, 0–100) və Siyasi hakimiyyət meyli (x oxu, −5.0…+5.0). Sosial-mədəni meyl ayrıca bal kimi təqdim edilir.

Etibarlılıq
Sadə sual: Bu məqalənin məlumatlarına əsaslanaraq vacib qərar verə bilərəmmi?
Etibarlılıq — faktlara söykənmə, yoxlanıla bilən mənbələr və obyektiv dilin ümumi nəticəsidir.
Qeyd: Mənbə şəffaflığı (istinadın açıq göstərilməsi) və mənbə keyfiyyəti (mənbənin etibarlılığı) fərqli anlayışlardır; etibarlılıq onların cəm təsiridir. Mövqe/esse yazılarında istinad az ola bilər — bu “pis yazı” demək deyil, sadəcə yoxlana bilən faktlar azdır.
Siyasi hakimiyyət meyli
Sadə sual: Məqalə hakimiyyəti necə təqdim edir — müsbət, mənfi, yoxsa neytral?
−5.0 kəskin müxalif çərçivə, 0 neytral, +5.0 güclü hökumətyönümlü çərçivə.
Bu, müəllifin “ümumi siyasi baxışı” deyil — konkret məqalədə görünən meyldir.
Sosial-mədəni meyl
Sadə sual: Məqalə sosial qrupları və dəyərləri necə təqdim edir?
Kimlik, dəyərlər və həyat tərzi mövzularında tərəfkeş dil/məzmun siqnalları.

Çərçivələmə (Framing)

Sadə sual: Hadisə necə təqdim olunur — problem, həll, məsuliyyət, ya qəhrəmanlıq çərçivəsində?
Çərçivələmə eyni faktın müxtəlif söz seçimi və vurğularla fərqli “baxış bucağı”ndan təqdim edilməsidir.

Məsələn: “Polis etirazçıları çökdürdü” (repressiv çərçivə) vs. “Polis qayda pozuntularının qarşısını aldı” (qanun-qayda çərçivəsi).

Ballar və şkalalar

  • Etibarlılıq (0–100): 0–19 çox aşağı, 20–39 zəif, 40–59 orta, 60–79 güclü, 80–100 yüksək.
  • Siyasi hakimiyyət meyli (−5.0…+5.0): −5 kəskin müxalif, 0 neytral, +5 güclü hökumətyönlü.
  • Sosial-mədəni meyl (−5.0…+5.0): Mənfi/pozitiv dəyər çərçivəsi və vurğular.

Qeyd: Ballar mətn daxilindəki siqnalların yığcam təsviridir; bu, müəllifin “tam siyasi mövqeyi” deyil.

Diaqnostik göstəricilər

Dilin yüklülüyü (0–100)

Sadə sual: Məqalə sizi məlumatlandırır, yoxsa daha çox emosiya oyadır?

Dilin yüklülüyü — neytral sözlər əvəzinə güclü emosional assosiasiyalı terminlərin istifadəsidir. Emosional dil təkbaşına manipulyasiya demək deyil; manipulyasiya faktlar gizlədildikdə və ya təhrif edildikdə yaranır.

  • 0–25: Neytral.
  • 26–50: Mülayim emosional ton (qəbul edilə bilən).
  • 51–75: Güclü emosional dil (faktları kölgədə qoya bilər).
  • 76–100: Çox yüklü/manipulyativ.

Mənbə şəffaflığı (0–100)

Sadə sual: İddiaların mənbələri açıq göstərilibmi və yoxlana bilərmi?

Şəffaflıq (istinadın görünməsi) və keyfiyyət (mənbənin etibarlılığı) fərqlidir. Paywall altındakı akademik məqalə yüksək keyfiyyətli, lakin məhdud şəffaf ola bilər; açıq xəbər agentliyi linki şəffafdır, lakin keyfiyyət ayrıca qiymətləndirilir.

Başlığın dəqiqliyi (0–100)

Sadə sual: Başlıq məzmunu düzgün əks etdirir, yoxsa klik üçün şişirdilib?

“Klikbeyt” sensasiyalı və yayındırıcı başlıq dilidir; informasiyadan çox emosiyaya fokuslanır.

Dil siqnalları

Sadə sual: Məqalədə hansı dil kateqoriyaları öndədir?

  • Yüklü sözlər: “faciə”, “xəyanət”, “möcüzə”.
  • Qeyri-müəyyən: “deyilir ki”, “bəziləri hesab edir”.
  • Mütləq: “həmişə”, “heç vaxt” — yalnız kontekstdən asılı olaraq problemli; məsələn, “Bütün insanlar həmişə ölür” elmi faktdır.
  • Spekulyativ: “ola bilər”, “ehtimal ki” — proqnozlarda normaldır; həddindən artıq istifadəsi zəiflik əlaməti ola bilər.
  • Yüklü sinonimlər: “rəhbər” vs “prezident” — bəzən birincisi müsbət assosiasiya yaradır; kontekstə diqqət yetirilir.

Qeyd: Bu göstəricilər istifadəçiyə şəffaflıq üçün ayrıca göstərilir və bal hesablanmasına daxil edilə bilər; hər birinin çəkisi model versiyasından asılıdır.

Məqalə növü (meta)

meta.article_type sahəsi məzmunu “xəbər”, “rəy”, “analitika”, “reportaj” və s. kimi təsnif edir. Rəy və şəxsi esselərdə istinad az ola bilər; bu, etibarlılıq balını texniki olaraq aşağı göstərə bilər, amma yazının dəyərini avtomatik azaltmır.

JSON çıxarış sxemi

Niyə JSON?

JSON — struktur, şəffaflıq və avtomatlaşdırma üçün standart formatdır. Nəticələr digər sistemlərə rahat inteqrasiya olunur.

Nümunə sxem (Azərbaycanca mətn tələbi ilə):

{
  "meta": {
    "article_type": "xəbər | rəy | analitika | reportaj | ... "
  },
  "scores": {
    "reliability": {
      "value": 0-100,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf. Vergüllərlə cümlələri ayırma."
    },
    "socio_cultural_bias": {
      "value": -5.0 ... 5.0,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf."
    },
    "political_establishment_bias": {
      "value": -5.0 ... 5.0,
      "rationale": "Aydın, axıcı Azərbaycan dilində paraqraf."
    }
  },
  "diagnostics": {
    "language_loadedness": 0-100,
    "sourcing_transparency": 0-100,
    "headline_accuracy": 0-100,
    "language_flags": [
      { "term": "…", "category": "…" }
    ]
  },
  "cited_sources": [
    { "name": "…", "role": "…", "stance": "…" }

Praktiki nümunələr

Nümunə 1: Yüksək etibarlılıq

Başlıq: “Maliyyə Nazirliyinin son hesabatına görə dövlət büdcəsinin kəsiri 2% artıb”.

Mətn: “... rüblük hesabat ... gəlirlər planlaşdırılandan 1.5% aşağı ...”

Qiymətləndirmə: Etibarlılıq 85/100; Mənbə şəffaflığı 90/100; Başlığın dəqiqliyi 90/100; Dilin yüklülüyü 15/100.

Nümunə 2: Qərəzli təqdimat

Başlıq: “Müxalifət yenə də xəyanətə davam edir!”

Mətn: “... Deyilənə görə ... Ekspertlər bildirirlər ki ...”

Qiymətləndirmə: Etibarlılıq 25/100; Siyasi meyl +4.0; Mənbə şəffaflığı 20/100; Dilin yüklülüyü 85/100; Başlığın dəqiqliyi 30/100.

Nümunə 3: Balanslaşdırılmış rəy

Başlıq: “Şərh: Yeni vergi siyasətinin müsbətləri və mənfiləri”.

Mətn: “... müxtəlif reaksiyalar ... hər iki tərəfin arqumentləri ...”

Qiymətləndirmə: Etibarlılıq 75/100; Siyasi meyl +0.5; Mənbə şəffaflığı 80/100; Dilin yüklülüyü 30/100.

Qrafiki necə şərh etməli

LNK.az-da istifadə olunan koordinat oxları yanaşması beynəlxalq praktikadan, əsasən də Ad Fontes Mediadan ilhamlanır, lakin yerli kontekstə uyğunlaşdırılıb. Yuxarı daha yüksək etibarlılıq, aşağı daha zəif; sol müxalif çərçivə siqnalları, sağ hökumətyönümlü siqnallar.

Tək nöqtə bütün mətnin ortalamasıdır; içindəki müxtəlif səslər və ton fərqləri “dil siqnalları” bölməsində görünə bilər.

Məhdudiyyətlər

  • Kontekst: İroniya, sarkazm və mədəni istinadlar səhv şərh edilə bilər.
  • Dil mürəkkəbliyi: Atalar sözləri və bədii ifadələr düzgün qiymətləndirilməyə bilər.
  • Aktuallıq: Model ən son xəbərləri dərhal bilmir.
  • Subyektivlik: Bəzi göstəricilər kontekstdən asılıdır.
  • Məqalə növü: Rəy/şəxsi esselərdə istinad az ola bilər; bu, etibarlılıq balını texniki olaraq aşağı göstərə bilər.

Etika prinsipləri

  • Şəffaflıq: metod, meyar və əsaslandırmalar açıqdır.
  • Müstəqillik: siyasi/korporativ asılılıq yoxdur.
  • Ədalət: bütün mənbələrə eyni standart tətbiq olunur.
  • Məxfilik: istifadəçi məlumatları qorunur, üçüncü tərəflərə ötürülmür.
  • Məsuliyyət: səhvlər etiraf olunur, düzəlişlərə açığıq.
