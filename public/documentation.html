Ümumi baxış

LNK.az media qərəzini və etibarlılığını təhlil etmək üçün geniş imkanlara malik API təqdim edir. API xəbər məqalələrini qiymətləndirmək və etibarlılıq göstəriciləri, siyasi qərəz analizi və mənbə istinadları daxil olmaqla ətraflı təhlil təmin etmək üçün qabaqcıl süni intellekt modellərindən istifadə edir.

Əsas URL: https://lnk.az/api

Təkmilləşdirici qeydiyyatı

API istifadəsi üçün təkmilləşdirici hesabı yaratmaq lazımdır. Qeydiyyat prosesi sadə və sürətlidir.

Qeydiyyat linki: Təkmilləşdirici qeydiyyatı

Qeydiyyat addımları

  1. Məlumatları doldurun: E-poçt ünvanı, şifrə və şirkət məlumatlarınızı daxil edin
  2. API istifadə məqsədini bildirin: Layihənizin məqsədini və API-ni necə istifadə edəcəyinizi izah edin
  3. Təsdiqləməni gözləyin: Admin tərəfindən təsdiqlənməsi üçün gözləyin (adətən 24 saat ərzində)
  4. API açarlarını əldə edin: Təsdiqləndikdən sonra təkmilləşdirici panelindən API açarlarını yarada bilərsiniz
Vacib: Qeydiyyat zamanı düzgün və etibarlı məlumatlar daxil edin. Yalnış məlumatlar təsdiqlənməni gecikdirə bilər.

Təkmilləşdirici paneli

Təsdiqləndikdən sonra aşağıdakı imkanlara malik olacaqsınız:

  • API açarları idarəetməsi: Yeni açarlar yaradın, mövcud açarları silin və ya yeniləyin
  • İstifadə statistikaları: API istifadənizi real vaxtda izləyin
  • Limitlərə nəzarət: Plan limitlərinizi və istifadə həddinizi görün
  • API sınağı: Daxili test aləti ilə API-ni sınayın
  • Ödəniş idarəetməsi: Plan yeniləməsi və ödənişləri idarə edin

Autentifikasiya

API istifadəsi üçün təkmilləşdirici hesabı və API açarları tələb olunur. API açarları təkmilləşdirici panelindən yaradıla bilər.

API açarı əldə etmə

  1. Təkmilləşdirici hesabınızı yaradın və təsdiqləndiyindən əmin olun
  2. Təkmilləşdirici panelinə daxil olun
  3. "API açarları" bölməsindən yeni açar yaradın
  4. Açarı təhlükəsiz yerdə saxlayın

API açarı istifadəsi

API sorğularında açarı aşağıdakı üsullardan biri ilə göndərin:

Header ilə (tövsiyə olunan)

curl -X POST https://lnk.az/api/analyze \ -H "Content-Type: application/json" \ -H "X-API-Key: your-api-key-here" \ -d '{"url": "https://example.com/news-article"}'

Query parametri ilə

curl -X POST "https://lnk.az/api/analyze?api_key=your-api-key-here" \ -H "Content-Type: application/json" \ -d '{"url": "https://example.com/news-article"}'
Təhlükəsizlik: API açarlarınızı heç vaxt kodunuzda açıq şəkildə saxlamayın. Mühit dəyişənləri və ya təhlükəsiz konfiqurasiya faylları istifadə edin.
Qeyd: API açarı olmadan sorğular IP ünvanına əsaslanan sürət limitlərinə tabedir. API açarı ilə plan limitlərinizdən istifadə edə bilərsiniz.

API açarı olmadan

API açarı olmadan sorğular IP ünvanına əsaslanan ümumi limitlərə tabedir:

  • Təhlil API-si: dəqiqədə 10 sorğu
  • Digər API-lər: məqbul istifadə gözlənilir

Limit aşımı

Sürət limitini aşdıqda aşağıdakı cavabları alacaqsınız:

  • 429 Too Many Requests: Dəqiqə limiti aşılıb
  • 429 Daily Limit Exceeded: Günlük limit aşılıb
  • 403 Forbidden: API açarı etibarsızdır və ya deaktivdir
Məsləhət: Limitlərinizi izləmək üçün təkmilləşdirici panelindən istifadə edin. Real vaxtda istifadə statistikalarınızı görə bilərsiniz.
Qeyd: Limit aşımı zamanı sorğular bloklanır. Yeni sorğu göndərməzdən əvvəl limitin sıfırlanmasını gözləyin.

Təhlil API-ləri

POST /api/analyze
Xəbər məqaləsini qərəz və etibarlılıq baxımından təhlil edin. Bu, əsas API ucnöqtəsidir.

Sorğunun gövdəsi

Parametr Növ Tələb olunur Təsvir
url string Yes Təhlil ediləcək məqalənin URL-i
modelType string No İstifadə olunacaq AI modeli: "auto", "flash-lite", "flash", "pro"

Nümunə sorğu

curl -X POST https://lnk.az/api/analyze \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com/news-article", "modelType": "auto" }'

Nümunə cavab

{ "hash": "abc123def456", "meta": { "article_type": "xəbər", "title": "Article Title", "original_url": "https://example.com/news-article", "publication": "example.com", "published_at": "2025-01-16T10:30:00Z" }, "scores": { "reliability": { "value": 85, "rationale": "Məqalə faktlara söykənir və mənbələr göstərilir..." }, "political_establishment_bias": { "value": 1.2, "rationale": "Məqalədə hakimiyyətə müsbət meyl var..." } }, "diagnostics": { "socio_cultural_descriptions": [...], "language_flags": [...] }, "cited_sources": [...], "human_summary": "Məqalənin əsas məzmunu...", "is_advertisement": false, "analyzed_at": "2025-01-16T10:35:00Z" }
GET /api/get-analysis?id={hash}
Əvvəl təhlil edilmiş məqaləni onun hash/ID-si ilə geri qaytarır.

Sorğu parametrləri

Parametr Növ Tələb olunur Təsvir
id string Yes Təhlilin hash/ID dəyəri

Məlumat və statistika API-ləri

GET /api/statistics
Platformanın istifadəsi üzrə statistikanı, o cümlədən ümumi təhlilləri və son aktivliyi əldə edin.

Nümunə cavab

{ "total_analyses": 1250, "recent_analyses": 15, "timestamp": "2025-01-16T10:35:00Z" }
GET /api/site-averages?host={hostname}
Müəyyən bir xəbər saytının etibarlılıq və qərəz statistikalarını əldə edin.

Sorğu parametrləri

Parametr Növ Tələb olunur Təsvir
host string Yes Host adı (məs., "oxu.az")
url string No Tam URL (host avtomatik çıxarılacaq)
GET /api/recent-analyses
Reklam olmayan ən son 5 təhlili əldə edin.

Köməkçi API-lər

Export API-ləri

PDF Export: PDF export funksionallığı çıxarılıb. Bunun əvəzinə "Çap Et" düyməsi istifadə olunur. Modern brauzerlər HTML səhifələrini PDF formatında çap edə bilər.
GET /api/export/csv?id={analysis_id}
Təhlil məlumatlarını CSV formatında yükləyin. Spreadsheet təhlili üçün uyğundur.

Nümunə sorğu

curl -X GET "https://lnk.az/api/export/csv?id=abc123def456" \ -H "X-API-Key: your-api-key-here" \ -o analysis-data.csv
GET /api/export/html?id={analysis_id}
Təhlil hesabatını HTML formatında yükləyin. Web paylaşımı üçün uyğundur.

Nümunə sorğu

curl -X GET "https://lnk.az/api/export/html?id=abc123def456" \ -H "X-API-Key: your-api-key-here" \ -o analysis-report.html
GET /api/export/bulk?format={format}&user_id={user_id}
İstifadəçinin bütün təhlillərini toplu olaraq export edin. CSV və ya HTML formatında.

Sorğu parametrləri

Parametr Növ Tələb olunur Təsvir
format string Yes Export formatı: "csv" və ya "html"
user_id string Yes İstifadəçinin unikal ID-si
limit number No Export ediləcək maksimum təhlil sayı (default: 100)
offset number No Başlanğıc pozisiyası (pagination üçün)

Nümunə sorğu

# Bütün təhlilləri CSV formatında export et curl -X GET "https://lnk.az/api/export/bulk?format=csv&user_id=user123" \ -H "X-API-Key: your-api-key-here" \ -o all-analyses.csv # Bütün təhlilləri HTML formatında export et curl -X GET "https://lnk.az/api/export/bulk?format=html&user_id=user123" \ -H "X-API-Key: your-api-key-here" \ -o all-analyses.html
Export formatları:
  • PDF: Professional hesabat formatı, çap üçün uyğundur
  • CSV: Spreadsheet təhlili üçün məlumat formatı
  • HTML: Web paylaşımı üçün formatlanmış səhifə
Qeyd: Export API-ləri təkmilləşdirici hesabı və API açarı tələb edir. Bütün export əməliyyatları istifadəçi autentifikasiyası ilə məhdudlaşdırılıb.

Səhv idarəetməsi

Bütün API-lər uyğun HTTP status kodlarını və JSON formatında səhv mesajlarını qaytarır:

Status Kodu Təsvir Nümunə Cavab
200 Uğur Təhlil məlumatı və ya tələb olunan informasiya
400 Səhv Sorğu {"error": "URL daxil edilməyib."}
404 Tapılmadı {"error": true, "message": "Not found"}
429 Limit Aşılıb {"error": "Çox sayda sorğu göndərilir."}
500 Daxili Səhv {"error": true, "message": "Internal error"}

JavaScript/Node.js

// API açarınızı təyin edin const API_KEY = process.env.LNK_API_KEY || 'your-api-key-here'; // Məqaləni təhlil et const response = await fetch('https://lnk.az/api/analyze', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': API_KEY }, body: JSON.stringify({ url: 'https://example.com/news-article', modelType: 'auto' }) }); if (!response.ok) { throw new Error(`API səhvi: ${response.status}`); } const analysis = await response.json(); console.log('Etibarlılıq:', analysis.scores.reliability.value); console.log('Qərəz:', analysis.scores.political_establishment_bias.value); // Sayt statistikalarını əldə et const siteStats = await fetch(`https://lnk.az/api/site-averages?host=oxu.az&api_key=${API_KEY}`); const stats = await siteStats.json(); console.log('Orta etibarlılıq:', stats.avg_rel);

// JavaScript ilə export funksiyaları async function exportAnalysis(analysisId, format) { const response = await fetch(`/api/export/${format}?id=${analysisId}`, { headers: { 'X-API-Key': API_KEY } }); if (!response.ok) { throw new Error(`Export səhvi: ${response.status}`); } const blob = await response.blob(); const url = window.URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = `analysis-${analysisId}.${format}`; a.click(); window.URL.revokeObjectURL(url); } // PDF export await exportAnalysis('abc123def456', 'pdf'); // CSV export await exportAnalysis('abc123def456', 'csv'); // HTML export await exportAnalysis('abc123def456', 'html'); // Bulk export async function exportAllAnalyses(userId, format) { const response = await fetch(`/api/export/bulk?format=${format}&user_id=${userId}`, { headers: { 'X-API-Key': API_KEY } }); const blob = await response.blob(); const url = window.URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = `all-analyses.${format}`; a.click(); window.URL.revokeObjectURL(url); }

Terminologiya qeydləri

Bu sənəddə istifadə olunan bəzi terminlər bizim təklif etdiyimiz Azərbaycan dilində neologizmlərdir:

  • Ucnöqtə (endpoint): API-nin müəyyən funksiyasını yerinə yetirən URL ünvanı
  • Geri-çağırış ünvanı (webhook): Server tərəfindən avtomatik olaraq məlumat göndərilən URL ünvanı
  • Mühit dəyişənləri (environment variables): Proqram təminatının konfiqurasiyası üçün istifadə olunan dəyişənlər
