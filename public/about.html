Haqqımızda

LNK.az — Azərbaycanın ilk süni intellektə əsaslanan media qərəzi analizatorudur. Məqsədimiz oxuculara təqdim edilən xəbərlərin etibarlılığını və meylini göstərmək, ictimaiyyətin obyektiv informasiya əldə etməsinə kömək etməkdir.

Necə işləyir?

  • İstənilən xəbər linkini daxil edirsiniz.
  • Sistem məqaləni təhlil edir və etibarlılıq, siyasi meyl, sosial-mədəni meyl kimi göstəriciləri hesablayır.
  • Nəticələr qrafik üzərində göstərilir; əlavə diaqnostika ilə birlikdə təqdim olunur.

Metodologiyamız ilə daha ətraflı tanış olmaq üçün Metodologiya səhifəsinə baxın.

Niyə LNK.az?

  • Şəffaflıq: Balların arxasındakı əsaslandırmalar açıq göstərilir.
  • Müstəqillik: Layihə heç bir media qurumuna və ya siyasi təşkilata bağlı deyil.
  • Texnoloji üstünlük: Beynəlxalq metodologiyalardan ilhamlanmış, yerli informasiya mühitinə uyğunlaşdırılmış alqoritm.
  • Təhlükəsizlik: İki mərhələli doğrulama (2FA) və müasir şifrələmə texnologiyaları ilə məlumatlarınızın qorunması.
  • İstifadəçi dostu: İstifadəçi paneli, analiz tarixçəsi və fərdi hesab idarəetməsi imkanları.

Platform Xüsusiyyətləri

LNK.az təkcə analiz aləti deyil, həm də tam funksional platformadır:

  • İstifadəçi Hesabları: Analizlərinizi saxlamaq və tarixçəni izləmək üçün fərdi hesab yarada bilərsiniz.
  • Təhlükəsizlik: İki mərhələli doğrulama (2FA) ilə hesabınızı qoruyun.
  • API İnterfeysi: Təkmilləşdiricilər üçün RESTful API ilə platformanı öz layihələrinizə inteqrasiya edin.
  • Analiz Tarixçəsi: Bütün analizlərinizi bir yerdə görün və idarə edin.
  • Analiz Yeniləməsi: 1 aydan köhnə analizləri yenidən təhlil etmək üçün "Yenilə" düyməsini istifadə edin.
  • Mobil Uyğunluq: Bütün cihazlarda optimal işləmə təcrübəsi.

Təhlükəsizlik və Məxfilik

Məlumatlarınızın təhlükəsizliyi bizim üçün prioritetdir. Platformamızda aşağıdakı təhlükəsizlik tədbirləri tətbiq edilmişdir:

  • İki Mərhələli Doğrulama (2FA): Hesabınızı əlavə təhlükəsizlik qatı ilə qoruyun.
  • Şifrələmə: Bütün məlumatlar müasir şifrələmə standartları ilə qorunur.
  • Məxfilik: Şəxsi məlumatlarınız üçüncü tərəflərlə paylaşılmır.
  • Təhlükəsiz API: API açarı idarəetməsi və istifadə məhdudiyyətləri.

Daha ətraflı məlumat üçün Məxfilik Siyasəti səhifəmizə baxın.

Missiyamız

Biz inanırıq ki, obyektiv informasiya azad cəmiyyətin təməlidir. LNK.az oxuculara düzgün məlumat seçməkdə dəstək olmaqla, medianın keyfiyyətinin artmasına və sağlam ictimai müzakirələrin formalaşmasına töhfə verir. Məqsədimiz hər hansı bir məqaləyə "yaxşı" və ya "pis" damğası vurmaq deyil. Əksinə, bu alət oxucuların daha tənqidi və məlumatlı media istehlakçısı olmasına yardım etmək üçün güclü bir köməkçi kimi hazırlanmışdır.

Yeniliklər: 19 oktyabr 2025-ci ildən etibarən LNK sistemi analizləri həmişəlik saxlayır. 1 aydan köhnə analizləri "Yenilə" düyməsi ilə yenidən təhlil edə bilərsiniz. Bu, məqalələrin yenilənməsi və ya dəyişməsi halında daha dəqiq nəticələr əldə etməyə imkan verir.
